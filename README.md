# PAMO
Personal Assistant Modular Optimization

Esta aplicación aún está en fase de desarrollo.

Instalación
=============

Para instalar el asistente son necesarias las siguientes librerías:

*   PyAudio
    *   La manera más sencilla de realizar la instalación en windows es desde pipwin con el comando  
        \-- pip install pipwin --  
        seguido de:  
        \-- pipwin install pyaudio --  
        
*   SpeechRecognition
*   Playsound
*   Pyttsx3
